# QtQrCodeGenerator

This application generates a QR code from user-input text using Qt 5.15 and QML.

## 🚀 Features

- 🧾 Generate QR code from text input
- 📸 Real-time QR code preview
- 📋 Copy QR code to clipboard
- 📄 Save QR code as PDF
- 💡 Simple and user-friendly interface

## 🛠️ Technologies Used

- Qt 5.15 / QML
- C++
